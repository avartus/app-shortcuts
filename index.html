UpGate Shortcuts

This repository contains shortcut files for the UpGate app, which help you manage your app usage time.

Available Shortcuts

App Timer

A shortcut that waits for a specified time, then returns to the home screen and opens the UpGate app.

Import App Timer Shortcut

If the button doesn't work, you can manually download the shortcut file and import it into the Shortcuts app.

How to Use

After importing the shortcut, you need to create an automation:

  1. Open the Shortcuts app
  2. Go to the "Automation" tab
  3. Tap "+" to create a new automation
  4. Choose "App" as the trigger
  5. Select the app you want to monitor (e.g., Instagram)
  6. Choose "Is Opened"
  7. Tap "Next"
  8. Tap "Add Action"
  9. Search for "Run Shortcut" and select it
  10. Choose the imported "App-Timer" shortcut
  11. Tap "Next"
  12. Turn OFF "Ask Before Running"
  13. Tap "Done" to save the automation

Technical Details

The shortcut performs the following actions:

  1. Waits for a specified time (3 minutes by default)
  2. Returns to the Home Screen
  3. Opens the UpGate app

For more information, see the README.
